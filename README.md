# 📊 HR Analytics Dashboard

An interactive **HR Analytics Dashboard** built using **Power BI** to analyze employee data and provide insights into workforce trends, attrition, salaries, and departmental performance.

---

## 📷 Dashboard Preview

![HR Analytics Dashboard](HR%20Analytics%20Dashboard.png)

---

## 🎯 Project Overview

This dashboard helps HR teams and business leaders monitor key workforce metrics through interactive visualizations and filters. It enables users to explore employee distribution, attrition patterns, salary analysis, and hiring trends for data-driven decision-making.

---

## ✨ Key Features

### 📌 KPI Cards
- 👥 Total Employees
- 📉 Attrition Rate
- 🚪 Employees Left
- 💰 Average Salary
- 🎓 Average Experience

### 📊 Interactive Visualizations
- Total Employees by Year
- Employees by Department
- Average Salary by Department
- Attrition by Department
- Gender Distribution

### 🎛️ Interactive Filters
- City
- Department
- Job Role
- Gender
- Attrition Status

---

## 🛠️ Tools & Technologies

- **Power BI**
- **Power Query**
- **DAX (Data Analysis Expressions)**
- **Microsoft Excel**

---

## 📈 Business Insights

- Analyze employee distribution across departments.
- Monitor attrition trends and employee turnover.
- Compare average salaries between departments.
- Visualize workforce demographics by gender.
- Explore hiring patterns over time using interactive filters.
